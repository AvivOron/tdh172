חיפוש שיר במאגר

א ב ג ד ה ו ז ח ט י כ ל מ נ ס ע פ צ ק ר ש ת


  • משוררים
    כמות המשוררים אשר מונה המאגר הינה משוררים עבריים.
    להורדת רשימת שמות כל המשוררים לחץ כאן.

  • שירים
    כמות השירים אשר מונה המאגר הינה שירים.
    להורדת רשימת שמות כל השירים לחץ כאן.
  1. פיתוח סקריפט איחזור
    1. כתיבת סקריפט Python אשר פותח את אתר פרוייקט בן-יהודה ומוצא את אלמנטי ה-HTML הרלוונטים על מנת לאחזר את רשימת המשוררים במאגר.
    2. עבור כל משורר, פתחנו את לינק האובייקט הרלוונטי ובתוכו מצאנו את רשימת השירים הרלוונטית.
    3. עבור כל שיר שמצאנו, פתחנו את הלינק וחיפשנו את הטקסט המקורי בעברית.
    4. כל טקסט שמצאנו, פירסרנו, הורדנו ממנו את כל הניקוד העברי, החלפנו תווי יוניקוד לא חוקיים ובעצם הכנו את הטקסט לתרגום לכתב ברייל.
    5. תרגום הטקסט עפ"י מילון הממפה ביו אותיות עבריות לאותיות ברייל.
  2. יצירת מסד נתונים
    1. יצירת MySQL Database על גבי שירות RDS ב-Amazon AWS.
    2. בניית מודל טבלאות מתאים.
    3. שמירת כל המידע מסקריפט הפייתון בטבלאות.
  3. פיתוח ממשק Web
    1. בניית אתר מבוסס PHP ו-Javascript השולף מידע ממאגר הנתונים שבנינו ומציג אותו בצורה נוחה.
    2. ביצוע תהליך Deployment לשירות ElasinBeanstalk ב-Amazon AWS.
  4. פיתוח סקריפט העשרה
    1. כתיבת סקריפט Python אשר עבור כל משורר ממסד הנתונים, יאתר ערך Wikipedia מתאים עפ"י חיפוש מספר התאמות מירבי של שיריו בדף הערך הנבחן.
    2. לאחר מציאת הדפים הרלוונטיים (בדיקה אנושית), שליפת שנות לידה ופטירה ותקציר חייו של המשורר ושמירת כל הערכים לטבלת המשוררים במסד הנתונים.
    3. שליפת שנות לידה ופטירה משופרת באמצעות OpenRefine ומאגר VIAF ושמירת הנתונים למסד הנתונים.
    4. עבודה עם ספריית פייתון חיצונית להתממשקות מול Wikipedia.
במסך זה ניתן לצפות בנתוני תהליך מיצוי המידע אודות המשוררים הממופים במאגר.
