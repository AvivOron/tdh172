כחלק מהקורס בו אנו לוקחים חלק, "מדעי הרוח הדיגיטליים" בהנחיית ד"ר יעל נצר באוניברסיטת בן-גוריון בנגב,
בחרנו לממש פרוייקט אשר ינגיש לקהילת העיוורים וכבדי הראייה בישראל את מאגר
השירה העברית של פרוייקט בן-יהודה בתרגום לכתב ברייל.

לאחר שלבי האיחזור, העיבוד והתרגום, מצאנו לנכון להעשיר את ה-Metadata במאגר ולהוסיף תצוגות שונות של המידע המועשר באתרנו,
במטרה להגיע להבנה מעמיקה יותר אודות השירה העברית, מקורותיה ומשורריה עצמם.

במהלך העבודה על הפרוייקט יישמנו את הטכניקות השונות והשתמשנו בכלים המגוונים שלמדנו לאורך הסמסטר בקורס.
כמו כן, נאלצנו להתמודד עם אתגרים רבים וללמוד טכנולוגיות וכלים נוספים על מנת להגיע לתוצר הסופי -
החל מכתיבת ועיצוב דפי Web וכלה בהרמת סביבת Deployment לאחסון האתר.

אנו מקווים כי בפרוייקט זה, הקלנו במידה מסויימת על האתגרים היום יומיים של קהילת העיוורים וכבדי הראייה בישראל.

 תודה לפרוייקט בן יהודה אשר המיר את התכנים לפורמט דיגיטלי ואיפשר לנו לממש פרוייקט זה.

  • משוררים

    להורדת רשימת שמות כל המשוררים לחץ כאן.

  • שירים
    כמות השירים אשר מונה המאגר הינה שירים.
    להורדת רשימת שמות כל השירים לחץ כאן.

  • TEI
    להורדת כל קבצי ה-TEI במערכת לחץ כאן.
  1. פיתוח סקריפט איחזור
    1. כתיבת סקריפט Python אשר פותח את אתר פרוייקט בן-יהודה ומוצא את אלמנטי ה-HTML הרלוונטים על מנת לאחזר את רשימת המשוררים במאגר.
    2. עבור כל משורר, פתחנו את לינק האובייקט הרלוונטי ובתוכו מצאנו את רשימת השירים הרלוונטית.
    3. עבור כל שיר שמצאנו, פתחנו את הלינק וחיפשנו את הטקסט המקורי בעברית.
    4. כל טקסט שמצאנו, פירסרנו, הורדנו ממנו את כל הניקוד העברי, החלפנו תווי יוניקוד לא חוקיים ובעצם הכנו את הטקסט לתרגום לכתב ברייל.
    5. תרגום הטקסט עפ"י מילון הממפה ביו אותיות עבריות לאותיות ברייל.
  2. יצירת מסד נתונים
    1. יצירת MySQL Database על גבי שירות RDS ב-Amazon AWS.
    2. בניית מודל טבלאות מתאים.
    3. שמירת כל המידע מסקריפט הפייתון בטבלאות.
  3. פיתוח ממשק Web
    1. בניית אתר מבוסס PHP ו-Javascript השולף מידע ממאגר הנתונים שבנינו ומציג אותו בצורה נוחה.
    2. ביצוע תהליך Deployment לשירות ElasticBeanstalk ב-Amazon AWS.
  4. פיתוח סקריפט העשרה
    1. כתיבת סקריפט Python אשר עבור כל משורר ממסד הנתונים, יאתר ערך Wikipedia מתאים עפ"י חיפוש מספר התאמות מירבי של שיריו בדף הערך הנבחן.
    2. לאחר מציאת הדפים הרלוונטיים (בדיקה אנושית), שליפת שנות לידה ופטירה ותקציר חייו של המשורר ושמירת כל הערכים לטבלת המשוררים במסד הנתונים.
    3. שליפת שנות לידה ופטירה משופרת באמצעות OpenRefine ומאגר VIAF ושמירת הנתונים למסד הנתונים.
    4. עבודה עם ספריית פייתון חיצונית להתממשקות מול Wikipedia.
    5. כתיבת שאילתות SPARQL מעל Wikidata והצגת התוצאות בצורות ויזואליות שונות.
  5. יצירת קבצי TEI
    1. יצירת קבצי TEI עפ"י התקן המתאים עבור כל השירים במערכת.
    2. מתן אפשרות הורדת TEI עבור שיר ספציפי.
    3. מתן אפשרות הורדת כל קבצי ה-TEI במערכת.
  6. תיוג מקומות והצגה על מפה
    1. הרצת מתייג על כלל הטקסטים במאגר וחילוץ מקומות.
    2. עבור מקומות שחולצו, איחזור קוארדינטות משירות API של Google Maps.
    3. הצגת המיקומים שחולצו על מפת FusionTable.
לאחר הרצת מתייג על תוכן השירים במאגר, החלטנו להציג את התוצאות על גבי מפת העולם בתצוגות שונות על מנת לתת מענה לשאלה:
מהם המקומות אשר משוררי השירה העברית חלמו וכתבו עליהם? התוצאות לא מפתיעות כלל...
